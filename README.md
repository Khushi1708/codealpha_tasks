# CodeAlpha Java Internship Projects

This repository contains the Java projects completed as part of the CodeAlpha Java Programming Internship. These projects demonstrate core Java concepts, Object-Oriented Programming (OOP), data structures, user interaction, and problem-solving skills.

## 🚀 Projects Included

### 1. Student Grade Tracker
A Java application designed to manage and analyze student academic performance.

#### Features
- Add student names and grades
- Calculate average marks
- Find highest and lowest scores
- Display complete student performance summary
- Uses Arrays/ArrayLists for data management

#### Concepts Used
- Arrays & ArrayLists
- Loops and Conditional Statements
- Methods
- OOP Principles

---

### 2. Stock Trading Platform
A simulation-based stock trading system that allows users to buy and sell stocks while tracking portfolio performance.

#### Features
- Display available stocks and prices
- Buy and sell stocks
- Manage user portfolio
- Track investment performance
- Transaction history management

#### Concepts Used
- Object-Oriented Programming
- Collections Framework
- Encapsulation
- Exception Handling
- File Handling (Optional)

---

### 3. Artificial Intelligence Chatbot
A Java-based chatbot that interacts with users and responds to common queries.

#### Features
- Interactive conversation interface
- Rule-based response system
- Frequently Asked Questions support
- User-friendly communication
- Extensible knowledge base

#### Concepts Used
- String Processing
- Conditional Logic
- OOP Concepts
- Basic NLP Techniques
- GUI Integration (Optional)
